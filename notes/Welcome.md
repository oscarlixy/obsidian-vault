# Welcome to your Obsidian Vault

This is your personal knowledge base.

## Quick Start

- Create new notes in the `notes/` folder
- Use `templates/` for reusable note templates
- Store images and files in `assets/`
- Track daily notes in `daily/`

## Getting Started with Obsidian

1. **Markdown**: Write in Markdown syntax
2. **Links**: Use `[[Note Name]]` to link between notes
3. **Tags**: Use `#tag` to organize content
4. **Search**: Use Ctrl/Cmd + P to quickly find notes

Happy note-taking!
